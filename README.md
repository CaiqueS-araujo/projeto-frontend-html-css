# 🌐 Projeto Frontend — HTML, CSS e JavaScript

Projeto de desenvolvimento frontend criado durante a **Residência em TIC da Serratec**, com foco em aplicar conhecimentos de HTML, CSS e JavaScript na prática.

## 🚀 Demo ao vivo

👉 [Acesse o projeto aqui](https://testando-conhecimento-front.vercel.app)

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📁 Estrutura do projeto
projeto-frontend-html-css/
├── css/          → Estilos da aplicação
├── js/           → Scripts JavaScript
├── pages/        → Páginas HTML
└── index.html    → Página principal
## 💡 O que aprendi

- Estruturação de páginas com HTML semântico
- Estilização e responsividade com CSS
- Interatividade com JavaScript
- Deploy de projetos com Vercel
